## Modal

The modal used in this project is not original code.
It was sourced from:
https://webdesign.tutsplus.com/css-only-modal--cms-107772t

## Reason for Use
Our current understanding of CSS did not allow us to build a modal from scratch ourselves.
Originally, we had planned to create 15 seperate recipe pages for each individual recipes.
However, the would have heavily slowed development progress.

By using a CSS-only modal, we were able to display recipe information in a pop -up, as a result, now without needing to generate redundant HTML pages, making this project development much more efficient.
